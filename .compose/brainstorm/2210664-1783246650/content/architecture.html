Arquitectura del Sistema

World Cup 2026 Predictions App — Nuxt 4 + Nuxt UI + SQLite

Stack Decidido

A

Nuxt 4 + Nuxt UI v4

Framework + componentes UI listos, Tailwind v4 incluido

B

SQLite + Drizzle ORM

better-sqlite3, WAL mode, migrations con drizzle-kit

C

@sidebase/nuxt-auth

JWT + cookies httpOnly, session management

D

worldcup26.ir API

Gratis, sin key, 104 partidos, scores en vivo

Subsistemas

1. Autenticación

Registro, login, JWT, sesiones, perfil de usuario

2. Partidos & Predicciones

104 partidos, hacer predicciones, sistema de puntos (3/1/0)

3. Bracket Eliminatorio

Visualización animada r32→r16→qf→sf→final, tachando perdedores

4. Leaderboard & Rankings

Tabla de posiciones, top jugadores, ranking por acierto

5. Estadísticas & Gráficos

Charts interactivos, precisión, rachas, heat map

6. Notificaciones Push

Alertas 2h/1h/30min antes, resultados con confetti

7. Tablas de Grupo

12 grupos A-L con standings en tiempo real

8. Logros & Badges

Sistema de gamificación con 8+ logros desbloqueables

9. Panel Admin

Dashboard, gestionar partidos, resultados, usuarios

10. PWA

Service worker, manifest, instalable en móvil
